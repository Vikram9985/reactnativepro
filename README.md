# Novaloid Inventory Management App

A React Native inventory and billing app built with Expo, NativeWind, Redux, and AsyncStorage.

## Features

- Add products with name, price, quantity, and unique barcode/QR code
- Scan barcodes/QR codes with camera support
- View product details on scan
- Add products to cart by scanning
- Complete billing and reduce inventory quantity
- Local storage persistence with AsyncStorage

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the Expo project:

```bash
npm run start
```

3. Run on a device or emulator:

```bash
npm run android
npm run ios
```

## Notes

- Uses `expo-barcode-scanner` for scanning
- Uses `@react-native-async-storage/async-storage` for local persistence
- Uses `@reduxjs/toolkit` + `react-redux` for state management
- UI styled with NativeWind classes
